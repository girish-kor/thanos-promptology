## CLI Tools (Install Once, Use Forever)

```bash
# Install all essential CLIs
npm install -g \
  vercel \
  railway \
  wrangler \
  supabase \
  turbo \
  prisma \
  @nestjs/cli \
  tsx \
  playwright \
  @anthropic-ai/claude-code

# Useful one-liners
npx npm-check-updates -u          # Update all dependencies
npx depcheck                       # Find unused dependencies
npx license-checker                # List dependency licenses
npx madge --circular src/          # Find circular imports
npx why-is-node-running            # Debug hanging processes
```
