# @thanos/admin

Admin dashboard application for the Thanos Promptology project.

## Getting Started

```bash
pnpm dev
```

The admin dashboard will open on [http://localhost:3002](http://localhost:3002).

## Project Structure

- `app/` - Next.js App Router
- `components/` - UI components specific to admin
- `features/` - Admin features (users, analytics, etc)
- `styles/` - Tailwind config and global styles

## Scripts

- `pnpm dev` - Start dev server
- `pnpm build` - Build for production
- `pnpm start` - Start production server
- `pnpm lint` - Run ESLint
- `pnpm type-check` - Type check with TypeScript

## Features

- User management
- Analytics dashboard
- Settings configuration
- Reports generation
